## Context

### Root cause (confirmed by code audit)

`BankingAgent.js` dispatches `banking-agent-result` custom event on success (line 1585), but `UserDashboard.js` has **no listener** for that event. The `accounts` state in `UserDashboard` is only refreshed by:
- Initial mount (`fetchUserData()`)
- 30-second interval polling (line ~226)
- `banking-agent-hitl-confirmed` event (HITL path only)

So after a successful transfer/deposit/withdraw via the agent, the balance tiles on the dashboard don't update until the 30-second poll fires.
