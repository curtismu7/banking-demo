## Tasks

### Task 1: Create ChaseTopNav.js component

**Description:** New React component that renders a horizontal top navigation bar with:
- Chase navy background (`--chase-navy`)
- White text
- Centered horizontal nav links (Home, Dashboard, Config, etc.)
- User greeting on right side
- Logout button
- Theme toggle button
- Role-switch button (if applicable to UserDashboard)

**Output:** `banking_api_ui/src/components/ChaseTopNav.js` (~150 lines)

**Implementation notes:**
- Accept props: `user`, `onLogout`, `currentPage` (for active link highlighting)
- Use `useTheme()` for dark mode support
- Use `useIndustryBranding()` for preset branding
- Import `BrandLogo.js` for the Chase/BX logo mark in nav
- Styling: inline + `ChaseTopNav.css` (created in Task 4)
- Reuse existing UI components where possible (buttons, links)
- Layout: flex row with `justify-content: space-between` for left/right sections

### Task 5: Create ChaseTopNav.css

**Description:** Styling for the new ChaseTopNav component.

**Files:** `banking_api_ui/src/components/ChaseTopNav.css`

**Output:** ~80 lines

**Implementation notes:**
- `.chase-top-nav` — main flex container, `background: var(--chase-navy)`, `color: white`, small shadow
- `.chase-top-nav__left` — logo + brand name section
- `.chase-top-nav__center` — navigation links
- `.chase-top-nav__right` — user actions (greeting, logout, theme toggle, role switch)

- Dark mode overrides (if needed) using `html[data-theme='dark']`
