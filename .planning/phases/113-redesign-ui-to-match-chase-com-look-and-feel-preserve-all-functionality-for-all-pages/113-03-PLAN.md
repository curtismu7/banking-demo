# Plan 03: Chase Theme CSS (Fonts + Shared Styling)

**Objective:** Create a centralized `chase-theme.css` file with system font stack and shared Chase overrides, then apply it to high-traffic admin pages (appShellPages, AuditPage, FeatureFlagsPage).

**Context:** @113-CONTEXT.md

---

## Tasks

### Task 1: Create chase-theme.css

**Description:** New centralized stylesheet with system fonts and shared Chase color/layout rules.

**Files:** `banking_api_ui/src/styles/chase-theme.css` (~120 lines)

**Output:** Imported in `index.js` after `App.css`

**Implementation notes:**
- **Font stack (body):**
  ```css
  body {
    font-family: -apple-system, BlinkMacSystemFont, 'Segoe UI', Roboto, Helvetica, Arial, sans-serif;
    font-weight: 400;
  }
  
  strong, b {
    font-weight: 600;
  }
  
  h1, h2, h3, h4, h5, h6 {
    font-weight: 700;
  }
  ```

- **Chase color variables (if not already in index.css):**
  - Primary: `--chase-navy`, `--chase-blue`
  - Neutrals: `--chase-white`, `--chase-light-gray`, `--chase-medium-gray`, `--chase-dark-gray`
  - (These should already be defined in `index.css` from Phase 85)

- **Shared component overrides** for high-traffic pages:
  - `.app-page-shell__hero` — Chase navy background, white text
  - `.app-page-shell__title` — Navy color, bold
  - `.app-page-toolbar` — Light gray background, Chase navy text
  - `.app-page-toolbar-btn` — Navy color, white on hover
  - `.app-page-card` — White background, subtle 1px border in `--chase-medium-gray`

- **Dark mode overrides:**
  ```css
  html[data-theme='dark'] {
    body { color: var(--dash-text); background: var(--dash-bg); }
    .app-page-shell__hero { background: var(--dash-surface-2); }
    .app-page-card { background: var(--dash-surface); border-color: var(--dash-border); }
  }
  ```
