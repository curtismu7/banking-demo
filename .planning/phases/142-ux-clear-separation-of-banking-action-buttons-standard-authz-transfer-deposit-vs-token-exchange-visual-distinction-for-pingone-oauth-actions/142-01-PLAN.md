  <action>
Add repeating diagonal stripe overlay to `.transfer-btn`, `.deposit-btn`, `.withdraw-btn` (and their `-submit-` variants if they exist separately).

**Implementation approach:**
- Use CSS `repeating-linear-gradient` as a second background-image layer over the existing red gradient
- Stripe properties:
  - Direction: 45deg (left-to-right diagonal)
  - Pattern: thin stripes (e.g., 2px stripes + 2px gap = 4px repeat)
  - Color: rgba(255, 255, 255, 0.25) — white at 25% opacity (creates "carved into" effect)
  - Result: 20-30% visible stripe pattern through the underlying gradient
  
**CSS rule template:**
```css
.transfer-btn {
  background:
    repeating-linear-gradient(
      45deg,
      transparent,
      transparent 2px,
      rgba(255, 255, 255, 0.25) 2px,
      rgba(255, 255, 255, 0.25) 4px
    ),
    linear-gradient(180deg, var(--app-primary-red-mid) 0%, var(--app-primary-red) 100%);
  /* ... rest of existing styles ... */
}
```

Repeat for `.deposit-btn`, `.withdraw-btn`, and any other standard action button variants (look for similarly-named classes in the existing 1068-1120 line range).
