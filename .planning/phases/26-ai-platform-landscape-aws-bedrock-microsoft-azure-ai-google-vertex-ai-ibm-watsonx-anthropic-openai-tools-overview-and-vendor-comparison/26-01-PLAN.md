    ### Tab 1 — id: 'aws', label: 'AWS'

    Content covers Amazon Web Services AI tools:

    **Header:** AWS — "Build with the broadest AI/ML portfolio on the planet"

    **Tools to cover (one ToolCard each):**
    - **Amazon Bedrock** — Category: Foundation Model API — Managed serverless API for 30+ foundation models (Anthropic Claude, Meta Llama, Mistral, Titan, etc.). No GPU management; pay-per-token. Supports RAG via Knowledge Bases, agents via Bedrock Agents, and fine-tuning.
    - **Amazon SageMaker** — Category: ML Platform — End-to-end ML lifecycle: data prep, training, deployment, monitoring. Supports custom models, distributed training, real-time and batch inference. JumpStart library for pre-trained models.
    - **Amazon Q** — Category: Enterprise AI Assistant — Generative AI assistant for developers (Q Developer, integrated in VS Code/JetBrains) and business users (Q Business for document Q&A). Fine-tunable on company data.
    - **Amazon Rekognition** — Category: Vision AI — Image/video analysis: object detection, facial recognition, content moderation, text in image. REST API, no ML expertise required.
    - **Amazon Comprehend** — Category: NLP — Entity recognition, sentiment analysis, topic modeling, key phrase extraction on text. Supports custom entity/classification models.
    - **Amazon Lex** — Category: Conversational AI — Build chatbots and voice bots with NLU. Integrates with Lambda for fulfilment. Powers Alexa.
    - **Amazon Kendra** — Category: Enterprise Search — Intelligent search backed by NLP. Connects to S3, SharePoint, Salesforce. Bedrock Knowledge Bases is the newer evolution.
    - **Amazon Polly / Transcribe / Translate** — Category: Speech & Language — Polly: text-to-speech. Transcribe: speech-to-text. Translate: 75+ language pairs.

    **Key note:** AWS Bedrock Agents support multi-step agentic workflows with tool use; integrate with Step Functions for orchestration. IAM-native auth — no separate identity layer needed for AWS-internal agents.
