  <action>
    Create `banking_api_ui/src/components/VercelConfigTab.js` as a functional React component.

    **Component signature:**
    ```jsx
    export default function VercelConfigTab() { ... }
    ```

    **State:**
    - `vars` — array from GET /api/admin/vercel-config
    - `loading` — bool
    - `error` — string|null
    - `saving` — Record<key, 'idle'|'saving'|'success'|'error'>
    - `edits` — Record<key, string> (pending edit values)

    **On mount:** `fetch('/api/admin/vercel-config', { credentials: 'include' })` → set `vars`. On error → set `error`.

    **Grouping:** Group `vars` into categories based on key prefix:
    - `PINGONE_` → "PingOne"
    - `MCP_` or key includes `MCP` → "MCP Server"
    - `AUTHORIZE_` or `PINGONE_AUTHORIZE_` → "PingOne Authorize"
    - `CIBA_` or `STEP_UP_` → "Step-Up / CIBA"
    - Listed in INTERFACES as App vars → "App"
    - Everything else → "Other"

    **Render for each category:**
    ```jsx
    <section>
      <h3 className="vc-tab__category">{categoryName}</h3>
      <table className="vc-tab__table">
        <thead><tr><th>Variable</th><th>Value</th><th></th></tr></thead>
        <tbody>
          {vars.map(v => <VarRow key={v.key} var={v} ... />)}
        </tbody>
      </table>
    </section>
    ```

    **VarRow sub-component (inline or separate):**
    - If `v.type === 'secret'` or `v.type === 'encrypted'`:
      - Display key name, `<span className="vc-tab__secret-badge">🔒 set</span>` if `v.hasValue`, else `<span>not set</span>`
      - No editing — show lock icon and "Manage in Vercel dashboard"
    - If `v.type === 'plain'` or `v.type === 'system'`:
      - Display key name, an `<input type="text" value={edits[v.key] ?? v.value ?? ''} onChange={...} />`
      - A "Save" button: `onClick` → PATCH `/api/admin/vercel-config/${v.key}` with `{ value: edits[v.key] }`, set saving state, show "Saved!" or error
    - `saving[v.key] === 'saving'` → button shows "Saving…" and is disabled
    - `saving[v.key] === 'success'` → brief "✓ Saved" indicator (clear after 3s)
    - `saving[v.key] === 'error'` → "⚠ Failed" indicator

    **Loading / error states:**
    - Loading: show `<div className="vc-tab__loading">Loading Vercel environment variables…</div>`
    - Error: show `<div className="vc-tab__error">{error}</div>` with a Retry button

    Import `VercelConfig.css` in the component.
