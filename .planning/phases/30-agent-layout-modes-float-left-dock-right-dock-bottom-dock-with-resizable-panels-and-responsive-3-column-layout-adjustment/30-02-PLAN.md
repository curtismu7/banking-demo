  <action>
    Create `SideAgentDock.js` as a new component (pattern matches `EmbeddedAgentDock.js`):

    **Props:** `{ user, onLogout, side }` where `side: 'left' | 'right'`

    **Width state:** `widthPx` initialized from localStorage key `side_agent_dock_width_px` (default 340). Min 280, max 520.

    **Collapsed state:** `collapsed` initialized from localStorage key `side_agent_dock_collapsed` (default false).

    **Resize logic:** `onMouseDown` on the resize handle bar starts a `document.mousemove` listener. On mousemove, compute new width from `clientX` (left: clientX, right: window.innerWidth - clientX). Clamp to [280, 520]. On `mouseup`, persist to localStorage. Remove listeners on cleanup.

    **DOM structure:**
    ```jsx
    <div
      className={`side-agent-dock side-agent-dock--${side}${collapsed ? ' side-agent-dock--collapsed' : ''}`}
      style={{ width: collapsed ? 0 : widthPx }}
    >
      {/* Resize drag handle — on opposite edge from side */}
      <div className="side-agent-dock__resize-handle" onMouseDown={handleResizeMouseDown} />

      {/* Collapse/expand tab */}
      <button className="side-agent-dock__collapse-btn" onClick={() => setCollapsed(v => !v)} type="button"
        aria-label={collapsed ? 'Expand agent sidebar' : 'Collapse agent sidebar'}>
        {side === 'left' ? (collapsed ? '›' : '‹') : (collapsed ? '‹' : '›')}
      </button>

      {/* Agent */}
      {!collapsed && (
        <div className="side-agent-dock__body">
          <BankingAgent mode="inline" user={user} onLogout={onLogout} splitColumnChrome={false} />
        </div>
      )}
    </div>
    ```

    **Body class side-effect:** `useEffect` adds `app-has-side-dock-${side}` class to `document.documentElement` while mounted (and to reflect collapsed state with `app-has-side-dock-${side}--open`), removes on unmount. This allows CSS rules to shift page content.
