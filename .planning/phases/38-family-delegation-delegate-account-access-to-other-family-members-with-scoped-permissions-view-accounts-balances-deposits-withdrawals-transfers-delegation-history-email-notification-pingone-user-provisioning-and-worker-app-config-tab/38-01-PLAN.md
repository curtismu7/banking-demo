Create `banking_api_server/services/delegationService.js` implementing the full delegation domain logic.

**Storage:**
- On Vercel (`process.env.VERCEL === '1'`): use an in-memory Map keyed by delegation `id`.
- On local/non-Vercel: use `better-sqlite3` at `data/persistent/delegations.db`.
- Schema (SQLite only):
  ```sql
  CREATE TABLE IF NOT EXISTS delegations (
    id TEXT PRIMARY KEY,
    delegator_user_id TEXT NOT NULL,
    delegate_user_id TEXT,
    delegate_email TEXT NOT NULL,
    delegator_email TEXT,
    scopes TEXT NOT NULL DEFAULT '[]',
    status TEXT NOT NULL DEFAULT 'active',
    granted_at TEXT NOT NULL,
    revoked_at TEXT
  );
  CREATE INDEX IF NOT EXISTS idx_del_delegator ON delegations(delegator_user_id);
  ```
- Use `crypto.randomUUID()` for `id` generation.

**Exports (all async):**

1. `grantDelegation({ delegatorUserId, delegatorEmail, delegateEmail, scopes })`
   - Validate: `delegateEmail` required; `scopes` must be non-empty array containing only values from
     `['view_accounts','view_balances','create_deposit','create_withdrawal','create_transfer']`.
     Return `{ ok: false, error: 'validation_error', message: '...' }` on invalid input.
   - Prevent self-delegation: if `delegatorEmail && delegatorEmail.toLowerCase() === delegateEmail.toLowerCase()`,
     return `{ ok: false, error: 'self_delegation', message: 'Cannot delegate to yourself.' }`.
   - Prevent duplicate active delegation: check existing active delegation from same delegatorUserId to same delegateEmail.
     Return `{ ok: false, error: 'duplicate_delegation' }` if found.
   - Look up delegate in PingOne: call `fetchPingOneUserByUsername(delegateEmail)` from `pingOneUserLookupService`.
   - If NOT found: provision new user — call `getManagementToken()` from `pingOneClientService`, then:
     - Fetch `firstPopulationId` via `fetchFirstPopulationId(token, apiRoot)` from `pingoneBootstrapService` where `apiRoot = https://api.pingone.${region}/v1/environments/${envId}`.
     - `POST ${apiRoot}/users` with body `{ email: delegateEmail, username: delegateEmail, name: { given: 'Family', family: 'Member' }, population: { id: firstPopulationId }, lifecycle: { status: 'ACCOUNT_OK' } }`.
     - On HTTP error: return `{ ok: false, error: 'provisioning_failed', message: err.response?.data?.message || err.message }`.
     - Extract `delegateUserId = response.data.id`.
   - If found: `delegateUserId = user.id`.
   - Create delegation record object:
     `{ id, delegatorUserId, delegatorEmail: delegatorEmail || '', delegateEmail, delegateUserId, scopes: JSON.stringify(scopes), status: 'active', granted_at: new Date().toISOString(), revoked_at: null }`.
   - Persist to SQLite or in-memory Map.
   - Send grant email (best-effort, non-blocking): call internal `_sendDelegationEmail(delegateUserId, 'grant', delegatorEmail)`.
     Wrap in `try/catch`; log error and continue — do not propagate failure.
   - Return `{ ok: true, delegation: <record with scopes as parsed array> }`.

2. `revokeDelegation(id, delegatorUserId)`
   - Look up record (SQLite: `SELECT ... WHERE id = ? AND delegator_user_id = ?`; Map: find by id + verify owner).
   - If not found or already revoked: return `{ ok: false, error: 'not_found' }`.
   - Update: `status = 'revoked'`, `revoked_at = new Date().toISOString()`.
   - Send revoke email (best-effort): `_sendDelegationEmail(record.delegate_user_id, 'revoke', record.delegator_email)`.
   - Return `{ ok: true }`.

3. `listDelegations(delegatorUserId)`
   - Return all records with `status = 'active'` and `delegator_user_id = delegatorUserId`.
   - Return as array of objects with `scopes` parsed from JSON string to array.

4. `getDelegationHistory(delegatorUserId)`
   - Return ALL records (active + revoked) for `delegatorUserId`, ordered by `granted_at DESC`.
   - Return as array of objects with `scopes` parsed from JSON string to array.

**Internal helper (not exported):**

`async function _sendDelegationEmail(delegateUserId, type, delegatorEmail)`
- If `!delegateUserId`: return silently.
- `const token = await getManagementToken()` — if fails, log and return.
- Build HTML email body:
  - type `'grant'`: Subject `"Super Banking — You have been granted account access"`,
    body informs delegate that `${delegatorEmail || 'a user'}` has granted them access to their accounts.
  - type `'revoke'`: Subject `"Super Banking — Account access revoked"`,
    body informs delegate that their access to `${delegatorEmail || 'the account'}` has been revoked.
  - Use the same inline-HTML email style as `emailService.js` (warm gradient header, clean body).
- `POST /v1/environments/${envId}/users/${delegateUserId}/messages` using PingOne message API pattern.
- Log success/failure; never propagate.

**Create `banking_api_server/routes/delegation.js`:**

```js
'use strict';
const express = require('express');
const router = express.Router();
const {
  grantDelegation, revokeDelegation, listDelegations, getDelegationHistory
} = require('../services/delegationService');

// GET /api/delegation — list active delegations for authenticated user
router.get('/', async (req, res) => {
  try {
    const delegations = await listDelegations(req.user.id);
    res.json({ delegations });
  } catch (err) {
    console.error('[delegation] GET error:', err.message);
    res.status(500).json({ error: 'internal_error', message: err.message });
  }
});

// GET /api/delegation/history — full history for authenticated user
router.get('/history', async (req, res) => {
  try {
    const history = await getDelegationHistory(req.user.id);
    res.json({ history });
  } catch (err) {
    console.error('[delegation] GET /history error:', err.message);
    res.status(500).json({ error: 'internal_error', message: err.message });
  }
});

// POST /api/delegation — grant a new delegation
router.post('/', async (req, res) => {
  const { delegateEmail, scopes } = req.body || {};
  const delegatorEmail = req.user.email || req.user.username || '';
  const result = await grantDelegation({
    delegatorUserId: req.user.id,
    delegatorEmail,
    delegateEmail,
    scopes: Array.isArray(scopes) ? scopes : [],
  });
  if (!result.ok) {
    const statusMap = {
      validation_error: 400,
      self_delegation: 400,
      duplicate_delegation: 409,
      provisioning_failed: 502,
    };
    return res.status(statusMap[result.error] || 400).json(result);
  }
  res.status(201).json(result);
});

// DELETE /api/delegation/:id — revoke a delegation
router.delete('/:id', async (req, res) => {
  const result = await revokeDelegation(req.params.id, req.user.id);
  if (!result.ok) {
    return res.status(404).json(result);
  }
  res.json(result);
});

module.exports = router;
```
