  Response: { delegations: [ { id, delegatorUserId, delegateEmail, scopes: string[], status: 'active', granted_at, revoked_at } ] }

POST /api/delegation
  Body: { delegateEmail: string, scopes: string[] }
  Response: 201 { ok: true, delegation: {...} }
           400 { ok: false, error: 'validation_error' | 'self_delegation', message: string }
           409 { ok: false, error: 'duplicate_delegation' }
           502 { ok: false, error: 'provisioning_failed', message: string }

DELETE /api/delegation/:id
  Response: 200 { ok: true } | 404 { ok: false, error: 'not_found' }

GET /api/delegation/history
  Response: { history: [ { id, delegateEmail, scopes, status, granted_at, revoked_at } ] }

```jsx
import React, { useState, useEffect, useCallback } from 'react';

export default function DelegationPage({ user }) { ... }
```

**State:**
```js
const [delegations, setDelegations]   = useState([]);
const [history, setHistory]           = useState([]);
const [loading, setLoading]           = useState(true);
const [error, setError]               = useState('');
// Add delegate form
const [delegateEmail, setDelegateEmail] = useState('');
const [selectedScopes, setSelectedScopes] = useState(['view_accounts', 'view_balances']); // default read-only
const [submitting, setSubmitting]     = useState(false);
const [submitError, setSubmitError]   = useState('');
const [submitSuccess, setSubmitSuccess] = useState('');
// Revoke
const [revoking, setRevoking]         = useState(null); // delegation id being revoked
// Tab
const [activeSection, setActiveSection] = useState('active'); // 'active' | 'history'
```

**loadData callback (useEffect on mount):**
- `GET /api/delegation` → setDelegations(data.delegations || [])
- `GET /api/delegation/history` → setHistory(data.history || [])
- Set loading false; catch and setError on failure.

**handleGrant:**
- Validate: email required, at least one scope selected.
- POST /api/delegation with { delegateEmail, scopes: selectedScopes }.
- On success: refresh loadData, clear form, setSubmitSuccess("Access granted to {delegateEmail}").
- On error: setSubmitError(data.message || "Grant failed").
- Clear success/error message after 4s.

**handleRevoke(id):**
- setRevoking(id)
- DELETE /api/delegation/{id}
- On success: refresh loadData.
- On failure: console.error, show brief error in UI.
- setRevoking(null).

**handleScopeToggle(key):**
- Toggle key in selectedScopes array.

**Layout:**
