<objective>
Build the React UI layer for multi-vertical demo mode. Create a VerticalContext that provides vertical-specific config (terminology, theme, scopes) to all components, and a VerticalSwitcher component for live switching between Banking, Retail, and Workforce modes.

Purpose: Let demo presenters switch verticals live during a presentation — the UI rebrands instantly without page reload, showing that the same PingOne + MCP architecture works across industries.

Output: VerticalContext.js, VerticalSwitcher.js, App.js integration.
</objective>
